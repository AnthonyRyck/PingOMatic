# Ping'O'matic

Une application qui ping automatiquement une liste de machine, d'IP ou de site toutes les 5 minutes.  
Elle écrite en .Net 4.7.2 et en WPF.  

## Test
Code couleur :
- Gris : pas encore testé
- Orange : en cours de test
- Vert : Ping OK
- Rouge : pas de Ping
